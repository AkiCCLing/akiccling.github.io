上海二次元游玩地点推荐

本文最后更新于：12 天前

文章内推荐的都是长期固定的地点，但其实 ACG 相关的 IP 会时不时在某地开展具体的限时活动，其限时活动具有时效性，所以没有具体的推荐地点，但我也会把该地曾经的一些限时活动列举出来（有些地方会不断地与不同的 IP 进行联动）。
文章更新时间：2023-12-22 03:25:18

# 美罗城

有时会有 IP 的联动，固定的场所有三家：吉卜力工作室授权店 & GoodSmile & JUMP SHOP

吉卜力工作室授权店：可以去逛逛，价格很梦幻，不建议购买

GoodSmile：手办厂，一些热门的大货都能看到，价格肯定不如网上，但可以看到实物

JUMP SHOP：里面的 IP 不是很了解（过段时间补充）

# 茑屋书店

附近的地铁离这里不算很近，如果时间不是很多的话就不要来了，时间多的可以来这里逛逛，店内主要以国外书籍为主，二次元以及游戏相关的书籍也有不少。




# 现代书店（静安嘉里中心）

杂志（Fami 通、Type-Moon）、轻小说、画集都有，但是价格太贵了




# 天天玩 X 秋葉原日系动漫中心

街机很多，湾岸 5DX+、湾岸 3DX+、头文字 D、maimai DX、SDVX 等

# 街机烈火

主要以街机为主，也有音游（下次更新补图）在网上找了张图，大部分都是这种摇杆街机

# 第一百货商业街

去南京路可以顺带看看，周边价格贵，同样的周边，在这里比 CP 现场贵 100%

# 百联 ZX 创趣场

万代魂（南京路步行街）& Animate
最近 Animate 也搬到了这里，用了 21 年拍的图，下次更新




# 外文书店

去的时候注意时间（10:00-18:30）

# 静安大悦城

最新的联动信息请关注微信公众号「上海静安大悦城」

没有固定的 IP，但是 Animate cafe (静安大悦城北座 N809-2) 会经常与不同的 IP 联动

# 百米香榭商业街

1F 是一些模玩店，2F 大部分都是牌店
等待补图

# 迪美购物中心

周边不少，但是没什么个人喜欢的东西


# 高达基地

下次一定

# 自由高达 & GUNDAM BASE

每晚外面的高达都有表演，可以动起来，商场里有一整个高达的店

# 不推荐的地方

# 上海香港三联书店

因为网上有提到所以特地去看了一下，不建议去，没什么二次元浓度
