上海二次元游玩地点推荐

文章内推荐的地方大部分都是长期固定的地点，但 ACG 相关 IP 时不时会在上海开展联名或者主题活动。
活动具有时效性，所以没有具体的推荐地点，但我会把该地曾经的一些限时活动列举出来（有些地方会不断与不同 IP 进行联动）。

# 大型商场

# 百联 ZX 创趣场

万代魂（南京路步行街），最近 Animate 也搬到了这里

里面的店铺很多 围成一个正方形，每一层顺着浏览完就可以上楼了

周边价格也不是太离谱，官方周边有不少价格与网上相同，但是会有不同店不同价的情况

# 美罗城

三月兽 (B1)

Bilibili Goods (B1)

JUMP SHOP (1F)

吉卜力工作室授权店 (1F)

可以去逛逛，但里面东西的价格很梦幻，不建议购买

GoodSmile (2F)

手办厂，一些热门的大货都能看到，价格肯定不如网上，但可以看到实物

IP 联动：(通常在 5F，之前举办过明日方舟和碧蓝航线展)

# 静安大悦城

主要的店都在 3F、6F、8F（时间充足的话可以把这三层都逛一遍）

没有固定的 IP，但是像 IPSTAR | 潮玩星球 (静安大悦城北座 N607)、Animate cafe (静安大悦城北座 N809-2) 会经常与不同的 IP 联动：

2023.12.13 - 2024.01.16 「孤独摇滚！ × animate cafe」in animate cafe
https://mp.weixin.qq.com/s/LeYRJcAj1emDZknksYrvHQ

2023.12.06 - 2024.01.24 「咒术回战 怀玉・玉折篇 × animate cafe」
https://mp.weixin.qq.com/s/7HXRb1n9GxPZHNo6zqHf8w

有时还会有快闪店：
2023.08.04 SPY×FAMILY 间谍过家家限定快闪店

最新的联动信息请关注微信公众号「上海静安大悦城」

# Lalaport (自由高达 & GUNDAM BASE)

晚上外面的高达会有表演，可以动起来，商场里还有一整个高达的店

# 迪美购物中心

周边不少，但是没什么个人喜欢的东西

# 图书音像店

# 外文书店

去的时候注意时间（10:00-18:30）

会有当季的新番书籍

# 现代书店（静安嘉里中心）

杂志（Fami 通、Type-Moon）、轻小说、画集都有，但是价格太贵了

# 上海书城 (未完工)

进了书城后直上二楼，有一块区域有许多轻小说，包括宫崎骏的一些电影的绘本

5F 单独有一家店

楼层内有一些零星的二次元书籍

6F 的 MORFAN 店里有一些黑胶和 CD：

# 街机厅

街机厅里也有不少与 ACG 相关的机台，比如赛车和音游系列，详细的街机厅列表请参考全国音游地图：https://map.bemanicn.com/。下面仅列几个我去过的机厅。

# 天天玩 X 秋葉原日系动漫中心

街机很多，jubeat、太鼓达人、头文字 D、GTDR、湾岸 5DX+、湾岸 3DX+、头文字 D、maimai DX、SOUND VOLTEX 等

具体参考：https://map.bemanicn.com/shop/262

# 街机烈火

主要以街机为主 (有很多传统街机摇杆机台)，也有很多音游 (比如 maimai DX、SOUND VOLTEX、jubeat、太鼓达人、GTDR 等)，音游区有很多人，可能需要排队；店里。

具体参考：https://map.bemanicn.com/shop/1

# 待更新

# 高达基地

地址：上海市浦东新区陆家嘴西路 168 号正大广场 F5
等待更新

# 不推荐的地方

# 上海香港三联书店

因为网上有人提到所以特地去看了一下，不建议去，没什么二次元浓度

# 茑屋书店

位置比较偏，而且附近的地铁离这里不近，如果时间不是很多的话就不要来了，时间多的可以来这里逛逛，店内主要以国外书籍为主，二次元以及游戏相关的书籍也有不少。

# 百米香榭商业街

里面的店太杂了，不值得专门去，如果在人民广场或者南京路可以路过看一下。
1F 是一些模玩店，2F 大部分都是牌店

# 第一百货商业街

去南京路可以顺路看看，周边价格贵，同样的周边，在这里比 CP 现场贵 100%
