B 站常见问题参考回复（第 6 版）2024-8-10

一些不确定的问题（手机）

手机需要提供手机型号，系统版本和 app 版本，app 版本在 app 内设置–关于哔哩哔哩查看，如果 app 不是最新请尝试更新到最新版，如果是最新版可以试试重装或清除缓存

更换绑定手机号和邮箱

参考回复：
如果能登录账号，并且已绑定有效且可以正常使用的邮箱，那么可以通过验证邮箱进行更换：https://passport.bilibili.com/account/security#/bindphone

如果手机号失效并且没有绑定邮箱/邮箱无法使用并且记得账号密码可以登录，可以参考此动态解绑手机号：
https://t.bilibili.com/359940408556675238

如果没有绑定邮箱并且忘记账号密码，可以尝试进行账号找回申诉：https://account.bilibili.com/appeal/home.html#/find

如果已绑定邮箱但失效了，可以尝试联系客服更换邮箱：https://t.bilibili.com/359939974764976759

已绑定的手机号失效后，进行申诉时一般都要求填写原手机号，如果忘记了会导致申诉大概率失败，建议通过其他渠道（例如询问以前打过电话、发过短信的人等）找到或想起相关手机号。

直播粉丝勋章改版

参考回复：
粉丝勋章点亮状态下，在勋章对应的直播间内完成升级任务中的任意一个任务（观看直播、投喂付费粉丝团灯牌、投喂电池），即可自完成升级任务的时间开始重新计算，维持点亮状态 3 天。

目前三连弹窗是由 UP 主设置的，可以联系 UP 主适当修改位置，，目前只能关闭弹幕来关闭个性化弹幕

推荐算法/首页推荐不好用

推荐算法的运行机制与您与视频的互动情况和您观看相关视频的关联度相关，如果您观看多个相关的视频并积极互动，耐心等待一段时间，之后推送机制可能会给您推送您喜欢的内容，感谢您的反馈;
也可以多使用分区，频道，搜索，关注列表来找到自己喜欢类型的视频（可以在我的-设置-推送设置-关闭推送我可能感兴趣的内容）

为什么一些视频的播放量显示为“-”？

参考回复：
一般播放量显示为“-”代表相关视频及其弹幕/评论中含有敏感/争议等内容，存在风险，被 B 站管理员隐藏了播放量，并且必须登录帐号（还可能限制账号等级/是否为大会员）才能浏览。

建议恢复已失效视频的相关信息

考虑到视频失效基本上都是因为各种违规和 UP 主自行删除导致的，不适合再展示相关的信息，可能还有法律风险上的考虑。

不过目前仍有显示已失效视频信息的方法，要用到第三方服务，如果有需要可以上网查找一下。

如何使账号强制下线？

参考回复：
目前客户端内已有相关功能，可以进入设置–安全隐私–登录设备管理页面中开启登录设备管理功能，目前此功能仅针对“哔哩哔哩”客户端登录有效（网页端无效），并且可以移除已通过验证的设备，设备被移除后上面登录的账号会被强制下线。如果想强制下线没有通过验证的设备上登录的账号的话可以更改一下密码，更改密码后在所有地方都会被强制下线。

为什么必须完成实名制才能投稿？

B 站自 2017 年 7 月 5 日起需要完成账号实名制（绑定中国大陆非虚拟运营商手机号，没有则需完成实名认证）才能投稿：https://www.bilibili.com/blackboard/smtgpc.html，自2017年9月29日起全面实施账号实名制（以绑定手机号的形式）：https://www.bilibili.com/read/cv17527/

如何修改账号用户名（不是昵称）？

参考回复：
目前账号用户名（不是昵称）不可修改，并且是重要的账号隐私信息，在一些申诉中可能会用到，请注意保护。

网页端使用 B 站搜索功能时网页显示“错误号:412 由于触发哔哩哔哩安全风控策略，该次访问请求被拒绝”

参考回复：
这是 B 站后台检测到（或得知）你当时使用的 IP 地址存在非法网络活动行为，所以进行了安全限制，因为国内 IP 地址一般不是固定的，可能是相关 IP 地址之前被人恶意利用过，针对此情况请联系在线客服说明情况，并提供自己的 IP 地址申请解除限制。

在 B 站贝壳账户中通过支付宝进行提现时，为什么明明自己的每日额度还没有用完，就提示今日限额已用完？

参考回复：
目前 B 站全站公用一个支付宝提现接口，有一个总额度，这个总额度是支付宝面向 B 站官方的，当 B 站全站的每日提现总金额达到一定程度时那个接口就不能用了，B 站全站的总额度会在每日的 0 点重置，当遇到此情况时可以等到第二天再进行提现。

如何查看账号注册时间？

参考回复：
如果连年月都忘记的话可以参考以下方法查看（注意以下方法属于个人研究，不是 B 站官方提供的正式服务，随时可能会失效）：

先登录账号，然后打开此链接：https://member.bilibili.com/x2/creative/h5/calendar/event?ts=0

从打开的网页中找到类似这样的内容（浏览器可以按 Ctrl+F 搜索“jointime”关键词）：”jointime”:xxxxxxxx（一串数字）

复制“jointime”后面的那一串数字，然后打开此网页在“时间戳”一栏粘贴刚才复制的数字：https://tool.lu/timestamp/

最后点击转换按钮就能得到对应的注册时间

【重要提醒】：账号的注册时间属于账号隐私信息，目前在一些 B 站申诉服务中需要验证输入的账号注册时间来判断申诉者是否为账号的合规持有者，请注意保护并不要随意泄露，必要时可以在完成操作后清除浏览器全部缓存、痕迹（包括 cookie）和清除系统剪贴板内容（重启系统即可）。

为什么我账号历史记录里无故多出多条未知记录？这些视频都没有看过。

参考回复：
请检查一下浏览器设置中是否开启了类似“预加载网页”、“预读 WAP”的功能，如果有开启的话浏览器会在某些情况下自动预加载网页，这样也会算进观看记录中，建议关闭相关功能并保持使用 7 天再看看。

如果没有开启相关功能则可能是账号被他人使用了，请检查一下近期是否有将账号出借给他人和账号绑定信息等关键信息是否被修改过和检查登录记录是否有异常和及时修改账号密码。

为什么以前被回复的评论、动态等稿件，最近才收到提醒消息？/明明有新回复却没有收到提醒？

参考回复：
可能是当时由于服务器后台压力较大出现了不稳定情况漏掉了/评论内容触发了二次审核，审核期间仅发布者自己可以看到，也不会有任何提醒，等后来检查/审核通过后系统自动补发了相关的提醒消息。

客户端播放视频出现黑屏、白屏、绿屏、灰屏、花屏等问题怎么办？

参考回复：
请试试升级客户端到最新、正式版和升级系统到最新、稳定版和清除客户端缓存和在客户端设置—播放设置中开启/关闭 https 播放和在播放设置—解码模式选择中切换解码模式和关闭系统自带的视频增强功能。

如果无法解决请当再次出现相关问题时点击画面右上角的三个点进行播放反馈，完成后点击客户端我的—联系客服—底部联系客服按钮转人工服务向在线客服反馈。

我是 B 站风纪委员会成员，但为什么一直接不到案子？一直提示案子已经被处理完了。

参考回复：
由于暑假等原因，近期风纪委在线人数高于以往水平，出现晚间风纪委出现“无新任务”的情况，目前已通过调整单个任务结算总人数等措施，确保任务覆盖更多风纪委员，如有任何疑问可通过“社区中心-B 站新风纪委建议反馈收集”或风纪委邮箱-“judgement@bilibili.com​”进行反馈

问一下个人空间里的关注、获赞数量等数据的更新周期是多久？

参考回复：
目前个人空间里的关注数、获赞数是实时更新，粉丝数、播放量、阅读量是今天更新昨天的数据，明天更新今天的数据。

为什么客户端首页推荐的视频的封面/标题和打开后的不一样？

参考回复：
这是正常的，目前客户端首页推荐中视频、专栏等稿件的标题、封面可以与实际不一样，可以由相关运营人员手动修改。

私信的自动回复功能如何开启？

参考回复：
目前自动回复功能需要粉丝量达到 1000 才能使用。

目前 B 站官方不禁止使用非官方抽奖工具的抽奖行为，但是不支持和不提倡这种抽奖行为，会对含有使用非官方工具的抽奖行为的稿件（例如视频、动态、评论等）进行限流。
